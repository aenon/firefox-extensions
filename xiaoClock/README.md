# Xiao Clock

A simple and small clock that sits in the toolbar. Ideal for full-screen use of Firefox.

## Installation

Install from [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/xiao-clock/).

## Features

- Displays time in the toolbar
- Supports 12/24 hour format toggle (right-click to switch)
- System light/dark mode support

## Usage

The clock displays in your Firefox toolbar. Right-click on the clock to toggle between 12-hour and 24-hour formats.
